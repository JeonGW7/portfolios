생성자 = 생성자 메서드
1.생성자는 클래스이름과 같다 
2.생성자는 리턴형이 없다. 필드(객체)초기화가 목적이다 
3.인수를 갖는다 -> 오버로드가 가능하다 
4.객체생성시 호출된다 
---------------------------------
5.생성자를 넣지 않으면 디폴트 생성자가 자동으로 만들어 진다
6.인자있는 생성자 존재하면, 디폴트 생성자는 자동으로 만들어지지 않는다
7.this: 자신의 객체를 지칭한다(많이사용함) -> this.전역변수 = 매개변수 
  this() : 자신의 클래스 내의 생성자를 호출한다 (사용하지 않는다)
8.super : 상위 클래스 객체를 말한다  (사용하지말것)
  super() : 상위 클래스 생성자 호출을 한다 
9.this()는 생성자 안에서 첫줄에 기재해야 한다
  super()도 생성자 안에서 첫줄 기재해야 한다
 따라서 this()와 super()를 동시에 사용할 수 없다
