1)DBMS을 설명하시오
-데이터베이스를 효율적으로 관리할 수 있는 시스템을 의미한다

2)데이터베이스란 무엇인지 설명하시오
-여러 사람에 의해 사용될 목적으로 관리되는 정보집합을 의미한다
ex)은행의 고객 데이터베이스 


3)논리모델링이란 무엇인지 설명하시오 
개념모델링으로 부터 업무영역의 업무데이터 및 규칙을 구체적으로 표현한 것이다
