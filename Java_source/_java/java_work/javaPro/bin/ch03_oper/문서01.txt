연산자
1. 산술 연산자

 + -  *(곱하기)  /(나누기)  %(나머지)
 
 정수/정수 = 몫
 정수/실수 = 실수 ( 자동 형변환은 : 연산시 자료형이 큰쪽으로 형변환된다)
 100/3.0
 
 ++  : 1씩증가
 --  : 1씩감소
 
 i++ : 후위연산(많이사용)
 ++i : 전위연산
 
 i--
 --i
 
 그러나 단독으로 연산할때는 1증, 1감소
 복합연산 할때 만 전위연산, 후위연산이 의미가 있다
 
int a=5;
int b=7;
int c = ++a + b--; //복합연산 13 
      ③  ①  ②  ④
