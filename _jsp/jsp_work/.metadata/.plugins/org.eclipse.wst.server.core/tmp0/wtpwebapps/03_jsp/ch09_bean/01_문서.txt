빈=빈즈=컴포넌트(Button,Label,TextField,,,) : 클래스 자료형

자바 빈즈 = 클래스
-자바빈즈는 자바를 소프트웨어 컴포넌트(부품)을 만들기 위한 기술이다
-DTO=VO, DAO(DB처리로직)

-자바빈즈를 사용하는 이유 
:디자인부분과 로직처리부분을 분리하여 작업할 수 있다
(front-end(웹화면에 보여주는 작업) / back-end(DAO작업))

SI : 개발
SM : 유지 보수
-프로그램 작성의 효율을 높이기 위해서 소프트웨어컴포넌트를 만들어서 사용한다  
-장점: 재활용(ex)awt에서 Button,TextField,...)
-일반적으로 웹 사이트는 디자이너와 프로그래머가 협력하여 개발하게 된다 
그런데 프로그래머가 JSP페이지의 이곳 저곳에 자바코드를 입력했을 때
디자이너는 이 페이지를 해석하기 어려워지고 
개발 효율 또한 매우 떨어지게 된다
그래서 등장하게 된것이 자바빈즈이다 

자바빈은 JSP페이지의 디자인 부분과 로직부분을 나눔으로써 복잡한 코드를 줄이고
프로그램의 효율을 높일 수 있다
전문 디자인 / front-end / back-end
