# 2022-mvc_teamporject
# 인터넷 강의 사이트 제작

#### Project execution period : 2022.03.03~2022.03.10

-----------------------
## 프로젝트 목표

- MVC 패턴의 이해
- 학습내용 재정리 및 적용
- 팀 프로젝트를 통한 의사소통 방법과 팀워크 단련

## 개발 환경
> OS: window10
> 개발 툴: Eclipse Luna
> DB: MySQL5.5
> Was: Apache Tomcat8.0
> 개발 언어: Java jdk1.8.0, Servlet/JSP, HTML, CSS, JavaScript,JQuery, Ajax
